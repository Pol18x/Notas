# Notas.java
## Autor
*Pol Fernández*
## Descripción
Programa que define un **conjunto de notas académicas** cuyas funciones son:
- Buscar la nota máxima.
- Confirmar si la nota máxima se repite en el conjunto.
- Calcular la media del conjunto de notas.
## Dependencias
1. Tener instalado **Java** en tu sistema operativo. Mínimo **Java 17**. [Descargar Java](https://www.java.com/es/download/)
2. Tener un **IDE** instalado. Recomendamos **Eclipse**. [Descargar Eclipse](https://eclipseide.org/)
3. Tener una **terminal** disponible en tu sistema operativo para ejecutar el programa.
## ¿Cómo ejecutar el programa?
1. **Desde la terminal**
   - En **Linux**, abre el buscador, busca terminal y ejecútalo.
   - En **Windows**, abre el buscador, busca Símbolo del sistema o Windows Powershell y ejecútalo.
   - Si vas a ejecutar el **.jar**, ejecuta el siguiente comando: `java -jar Notas.jar`
   - Si vas a ejecutar el **.java**, ejecuta los siguientes comandos:
     1. `javac Notas.java` *Para compilar el archivo*
     2. `java Notas` *Para ejecutar el programa*
2. **Desde Eclipse**
   - Abre Eclipse y crea un nuevo proyecto en Java.
   - Copiar o mover `Notas.java` al proyecto
   - Listo, podrás ver el código fuente y ejecutarlo desde el propio IDE.
